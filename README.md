# Rails Basic Blog

## Ruby Version
2.6.5

## Rails Version
6.0.2.1

## Setup

* Clone project
* run `bundle`
* run `rake db:migrate db:seed`
* run `rails test`

## Completed Features

* Display public posts
* Display public posts by blog
* Display a post

## Controller and Model Tests Coverage

* Blogs
* Users
* Posts

## Development Approach

Test Driven Development (TDD)
